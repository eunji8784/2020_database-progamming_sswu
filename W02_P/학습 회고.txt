#새로 배운 내용

1. PHP는 웹서버와 데이터베이스 사이에서 서로 통신이 가능하도록 서로의 언어를 해석하고 번역해준다.
2. 데이터베이스와 연동하는 PHP 함수
- mysqli_connect: 데이터베이스와 연결 
- mysqli_query: MYSQL 쿼리 실행
- mysqli_fetch_array : mysqli_query 를 통해 얻은 리절트 셋(result set)에서 레코드를 1개씩 반환

+ Node.js와 express를 사용해서 웹서버를 만들어본 적은 있지만 PHP와 apache를 사용해서 웹을 구성해본 적은 처음이라 처음 배우는 내용도 많고 새로웠다.
특히 PHP 문법과 PHP가 어떤 식으로 웹 동작을 처리하는지에 대해서 알게되었고, PHP가 자동으로 html 문법으로 해석해주는 걸 보고 참 신기하고 편리하다는 생각이 들었다.
그리고 windows terminal이라는게 있는지도 처음 알았다. 항상 리눅스와 윈도우에서의 명령어가 너무 헷갈려서 기존의 cmd 창에서 프로그래밍할때는 불편한 점이 많았는데,
windows terminal은 훨씬 인터페이스도 깔끔하고 리눅스 명령어도 처리해줘서 편한 것 같다. 
앞으로는 주로 이 터미널을 통해 프로그래밍을 많이 하게될 것 같다.
